## 本地运行

您也可以克隆示例项目到本地运行：

```bash
# 克隆项目
git clone https://github.com/your-username/jn-axios.git

# 进入示例目录
cd jn-axios/examples

# 安装依赖
yarn install

# 启动开发服务器
yarn dev
```
